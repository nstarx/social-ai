# Social AI

Global AI & Prompt Engineering Readiness Reports.

## Overview

A collection of interactive reports and analyses examining global readiness for prompt engineering and AI coding agents in software development. Includes country-specific breakdowns and comparative data.

## Contents

- `index.html` -- Main landing page
- `global-ai-readiness-report.html` / `.md` -- Full global readiness report
- `Global_Readiness.html` -- Global readiness dashboard
- `world-ai-simulator.html` -- Interactive world AI simulator
- Country reports: `india.html`, `italy.html`, `romania.html`, `spain.html`
- AI model outcome files: `claude-outcome.txt`, `gemini-outcome.txt`

## Usage

Open `index.html` in a web browser to navigate the reports and interactive visualizations.
